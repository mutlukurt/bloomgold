# Bloom Gold - Luxury Jewelry E-Commerce

Bloom Gold is a premium, high-end e-commerce application designed for a luxury jewelry brand. It features a sophisticated design system, seamless shopping experience, and fully responsive layout optimized for all devices.

![Project Preview](src/assets/hero-bg.jpg)

This project is built with a modern, performance-focused tech stack:

### Core Framework
-   **[React 18](https://reactjs.org/)**: The library for web and native user interfaces.
-   **[Vite](https://vitejs.dev/)**: Next Generation Frontend Tooling for ultra-fast development and building.

### Styling & Design
-   **CSS3 Custom Properties (Variables)**: For a consistent and easily maintainable design system (colors, spacing, typography).
-   **Responsive Design**: Mobile-first approach ensuring a flawless experience on phones, tablets, and desktops.
-   **[Lucide React](https://lucide.dev/)**: Beautiful, consistent icon set.
-   **Google Fonts**: Integrating 'Playfair Display' for headings and 'Inter' for body text for a luxury aesthetic.

### State & Routing
-   **React Context API**: For global state management (Cart, Products, User interactions).
-   **[React Router DOM](https://reactrouter.com/)**: For client-side routing and navigation.

### Features
-   **Dynamic Product Catalog**: Filtering and category management.
-   **Shopping Cart**: Fully functional cart drawer with add/remove/update logic.
-   **Responsive Navigation**: Custom hamburger menu interaction for mobile devices.
-   **Legal Pages**: Privacy Policy and Terms of Service implementation.
